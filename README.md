# Inventory Manager
---
Proyecto en realización con el objetivo de permitir a un técnico de soporte llevar un reistro del inventario de su empresa, permitiendo administrarlo y reformarlo dinámicamente.

---
# Tecnologías utilizadas
## Frontend:
- TypeScript/Angular
- Tailwind CSS

## Backend:
- Java/Spring Boot
- JPA
- MySQL

## Despliegue:
- Docker
